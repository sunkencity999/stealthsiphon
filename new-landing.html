Key Features

Client-Side Scraping

Extract data directly in the browser with multiple export formats (JSON, CSV, TXT, HTML)

Built-in Proxy Server

No reliance on external CORS services, making your scraping more reliable and private

Headless Browser

Scrape JavaScript-rendered content with Puppeteer and automatic fallback mechanisms

Database Export

Export scraped data directly to SQLite, MongoDB, or MySQL databases

Scheduled Scraping

Set up recurring scraping tasks with a user-friendly interface - no cron knowledge required

Deep Link Following

Automatically crawl and scrape linked pages up to 3 levels deep for comprehensive data collection

Anti-Detection Measures

Bypass common anti-scraping techniques with user-agent rotation, request delays, and more

Installation

1

Clone the Repository

git clone https://github.com/yourusername/stealthsiphon.git
2

Install Dependencies

cd stealthsiphon
npm install
3

Start the Server

node server.js

Access Stealth Siphon at http://localhost:4200

4

Or Use the Installer

curl -s https://raw.githubusercontent.com/yourusername/stealthsiphon/main/install.sh | bash
